# GoldView - 极简国际金价悬浮窗与均价计算器

GoldView 是一款专为黄金投资者设计的桌面辅助工具。它提供实时的国际伦敦金价悬浮窗显示，以及一个专业的持仓均价计算器。

## 主要特性

- **实时金价悬浮窗**
  - **极简设计**：半透明毛玻璃效果，适配现代桌面风格。
  - **智能隐藏**：靠近屏幕边缘自动隐藏，鼠标悬停即刻显示。
  - **高频更新**：支持 0.5 秒频率的数据刷新。
  - **多屏支持**：自动识别多显示器环境，支持吸附到任意屏幕边缘。
- **均价计算器**
  - **轻拟物卡片化 UI**：专业的金融工具视觉风格。
  - **智能校验**：内置输入合法性检查。
  - **历史记录**：支持保存最近 5 条计算记录，方便对比。
  - **结果持久化**：计算结果保留两位小数，记录可随时折叠。
- **极致便携**
  - 提供绿色免安装版，一键运行。
  - 内置一键打包脚本，方便分发。

## 安装与运行

### 1. 运行绿色版
直接运行 `dist/GoldView.exe` 即可。

### 2. 源码运行
确保已安装 Python 3.8+ 环境，然后安装依赖：
```bash
pip install -r requirements.txt
```
运行主程序：
```bash
python gold_view.py
```

## 开发与打包

如果你想自己打包应用，可以使用内置的快捷脚本：
- **Windows**: 双击运行 `package.bat`。

该脚本会自动清理旧文件并生成极致压缩的单文件可执行应用。

## 技术栈

- **GUI 框架**: PyQt6
- **网络请求**: Requests
- **数据源**: 新浪财经 (HF_XAU), Gold-API (备用)
- **打包工具**: PyInstaller
